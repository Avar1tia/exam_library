# Экзаменационное задание — АИС «Электронная библиотека»

Реализовано основное задание и индивидуальный вариант 4, соответствующий
номеру 8 в группе.

## Возможности

- реестр книг и жанров;
- обложки с вычислением MD5;
- Markdown-описания и рецензии с очисткой Bleach;
- роли администратора, модератора и пользователя;
- создание, просмотр, редактирование и каскадное удаление книг;
- рецензии с оценкой от 0 до 5;
- пагинация списка книг;
- популярные книги за последние три месяца;
- пять недавно просмотренных книг для каждого посетителя;
- ограничение до 10 учитываемых просмотров одной книги в день;
- административный журнал просмотров;
- статистика просмотров аутентифицированных пользователей;
- фильтр статистики по датам;
- CSV-экспорт полного журнала и статистики.

## Установка и запуск

```powershell
cd "C:\Users\Administrator\Documents\New project\exam_library"
python -m venv ve
ve\Scripts\activate
pip install -r requirements.txt
python seed.py
python -m flask --app run:app run
```

Откройте <http://127.0.0.1:5000/>.

## Тестовые пользователи

| Роль | Логин | Пароль |
|---|---|---|
| Администратор | `admin` | `Admin123` |
| Модератор | `moderator` | `Moderator123` |
| Пользователь | `user` | `User12345` |

## Права

- Администратор создает, редактирует и удаляет книги, пишет рецензии и открывает статистику.
- Модератор редактирует книги и пишет рецензии.
- Пользователь пишет рецензии.
- Неаутентифицированный посетитель открывает главную страницу, страницу входа и страницы книг.
